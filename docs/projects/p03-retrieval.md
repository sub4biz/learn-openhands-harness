# P03: Retrieval

## What You Do

Run a prompt with terminal and file editing tools only, then run it with an MCP code-search server attached. Measure when extra retrieval earns its slot.

## Harness Mechanism

Lexical baseline versus lexical plus MCP `search_code`.
